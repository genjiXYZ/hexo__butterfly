# 段落

```md

```

## 字体

```md
*斜体文本*
**粗体文本**
***粗斜体文本***
```

*斜体文本*
**粗体文本**
***粗斜体文本***

最常用的是粗体

## 分隔线(没有感情的分隔线)

```md
***
* * * 
*****
- - - 
-----
我是看不出哦区别 我常用 ----- 好理解
```

***
* * *
*****
- - -
-----

## 删除线(打脸专用)

```md
~~打脸了~~
```

~~打脸了~~

## 下划线

```
<u>带下划线文本</u>
```
